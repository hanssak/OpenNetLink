//////////////////////////////////////////////////////
//AppEnvSetting.json 파일에 각 값별 기능을 설명한다.
//////////////////////////////////////////////////////

bUseFileCheckException : 파일검사 예외신청 메뉴를 보일지 여부
bShowAdminInfo : 초기 웰컴 대시보드에서 연락처를 표시할지 여부
strLanguage : 사용하는 언어
tScreenTime : 화면잠금 시간(분)
bScreenLock : 화면잠금 사용여부
bEmailApproveUse : 메일전송, 메일결재 메뉴 사용여부
UpdatePlatform : 업데이트 서버 OS 타입
UpdateSvcIP : 업데이트 서버 IP
bDenyPasswordZIP : 압축파일 전송 금지(true:금지, false:허용)
